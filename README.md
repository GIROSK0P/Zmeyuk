# Zmeyuk
Классическая игра "Змейка" в консоли на языке Rust

Поле отрисовывается с помощью эмодзи поэтому версию .rs стоит запускать либо из PowerShell, либо из любой другой консоли поддерживающей эмодзи(например консоль VSCode)

В версии .exe хуже графика, ведь она сделана из символов читаемых стандартной консолью.
